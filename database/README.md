## 🎯 学习路线

```
1. MySQL 基础
   ↓
2. 索引原理（B+ 树、聚簇索引）
   ↓
3. 事务与锁（隔离级别、MVCC）
   ↓
4. SQL 优化（执行计划、慢查询）
   ↓
5. Redis 基础（5 种数据结构）
   ↓
6. Redis 高级（持久化、集群、缓存策略）
```

---

## 💡 学习建议

- **理解原理**：索引为什么用 B+ 树？事务隔离级别解决了什么问题？
- **动手实践**：EXPLAIN 分析 SQL，慢查询日志优化
- **对比记忆**：MySQL vs Redis，不同隔离级别的区别
- **实际场景**：结合项目中的数据库使用经验
