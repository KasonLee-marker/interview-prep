## 🎯 学习路线

```
1. 分布式理论基础（CAP、BASE）
   ↓
2. 分布式锁（Redis、ZooKeeper）
   ↓
3. 分布式事务（2PC、3PC、TCC、Saga）
   ↓
4. 一致性协议（Raft、Paxos、ZAB）
   ↓
5. 服务治理（注册中心、配置中心）
   ↓
6. 消息队列（Kafka、RocketMQ）
```

---

## 💡 学习建议

- **理解权衡**：CAP 定理的取舍，一致性与可用性的平衡
- **对比方案**：不同分布式锁方案的优劣
- **实战经验**：结合项目中的分布式场景
- **跟进前沿**：Service Mesh、Serverless 等新趋势
