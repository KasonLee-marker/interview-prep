## 🎯 使用指南

### 按面试准备阶段

| 阶段 | 重点 | 建议时间 |
|------|------|---------|
| **基础复习** | 后端基础 + 数据库 | 2-3 周 |
| **进阶提升** | 分布式系统 + 算法题 | 3-4 周 |
| **冲刺模拟** | 系统设计 + HR 面试 | 1-2 周 |

### 按公司类型

| 公司类型 | 重点分类 |
|---------|---------|
| **大厂（字节/阿里/腾讯）** | 全部，重点：系统设计 + 算法 |
| **中型公司** | 后端基础 + 数据库 + 分布式 |
| **创业公司** | 后端基础 + 系统设计（偏实战） |

## 📌 学习建议

1. **理解优先**：不要死记硬背，理解原理最重要
2. **结合实践**：每个知识点想想实际项目中的应用
3. **定期复习**：用 Anki 或类似工具做间隔重复
4. **模拟面试**：找朋友互相提问，练习表达
