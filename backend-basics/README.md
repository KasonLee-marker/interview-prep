## 💡 学习建议

- **理解原理**：不要只记结论，理解为什么这样设计
- **结合代码**：每个知识点都能写出示例代码
- **对比记忆**：相似概念对比学习（如令牌桶 vs 漏桶）
- **实际场景**：思考在项目中的应用场景
